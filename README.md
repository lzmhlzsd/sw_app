## 项目生成

## 项目目录
```
├── www                                  
    ├── component                        // 页面逻辑
    ├── css                              // 公共库样式
    ├── img                              // 图片
    ├── include                          // 静态页面公共部分
    ├── js                               // 公共js
    ├── libs                             // 第三方库
    ├── views                            // 页面视图
```
